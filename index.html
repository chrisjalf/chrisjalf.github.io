Welcome to Astronomer

Overview

Astronomer consists of two parts, the frontend (React w/ Typescript) and the backend (NestJS). To make sure things work, it is advisable for you to install the latest version of npm

Go To Repository

Astronomer Frontend

After cloning the project, go inside 'astronomer' or run 'cd astronomer'

cd astronomer

Then. go inside 'astronomer-fe' or run 'cd astronomer-fe'

cd astronomer-fe

Once inside the directory, run the following command to install dependencies

npm install

Before running the project, run the following command to install Vite. The application runs using Vite

npm install -D vite

Run the following command to start the project

npm run dev

If you're unsuccessful in starting the project due to an error with description of 'rollup-darwin-x64', try running the following command before starting the project again

npm install @rollup/rollup-darwin-x64

Astronomer Backend

After cloning the project, go inside 'astronomer' or run 'cd astronomer'

cd astronomer

Then. go inside 'astronomer-be' or run 'cd astronomer-be'

cd astronomer-be

Once inside the directory, run the following command to install dependencies

npm install

Run the following command to start the project

npm run start:dev
